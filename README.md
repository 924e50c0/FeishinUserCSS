# FeishinUserCSS
UserCSS for Feishin.
Compatible with [Stylus](https://github.com/openstyles/stylus).

## Installation
Install or download the [theme](https://github.com/924e50c0/FeishinUserCSS/raw/refs/heads/main/feishin-theme.user.css) directly from GitHub or visit the [userstyles.world](https://userstyles.world/style/27093) page.

## Features
Initially made to emulate my musicbee setup.

- Full width horizontal visualizer
- Player bar at the top & seeker bar adjustments
- Filters overhaul
